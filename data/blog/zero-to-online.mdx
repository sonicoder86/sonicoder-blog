How do you put your static site online? 10 free hosting providers

Once you've finished coding your fantastic idea, it's time to show it off to the world.
But to do that, you need a place to host it online.
Spending a dollar on hosting services is unnecessary, as several companies offer free hosting plans.
These providers allow you to quickly get your website online without time limits or the need to charge your credit card.
In this article, I'll show you 10 providers that offer free hosting services (forever free).

### [Render.com](https://render.com/) (backend, frontend)

Render.com is a cloud platform offering hassle-free hosting and scaling solutions for static websites, APIs, and web services.

- Install Githubb App, authorize

### [Netlify](https://netlify.com/) (backend, frontend)

Netlify is a powerful platform that provides seamless deployment and hosting for static websites. Its features include continuous deployment, serverless functions, and form handling.

- Install Github App, authorize

### [Vercel]((https://vercel.com/)) (backend, frontend)

Vercel is a cutting-edge platform specializing in fast and efficient deployment and hosting for static websites, serverless functions, and Jamstack applications.

- Install Github App, authorize

### [Fly.io](https://fly.io/) (backend, frontend)

Fly.io is a versatile platform offering global hosting and edge computing solutions. It empowers developers to easily deploy and run applications closer to their users.
Fly.io transforms containers into micro-VMs that run on our hardware in 30+ regions on six continents.

https://dev.to/w3ndo/just-2-commands-and-the-app-is-live-flyio-is-amazing-1cae

https://fly.io

- Install fly cli
- `fly auth login`
- `fly launch --now`

### [Adaptable](https://adaptable.io/) (backend)

Adaptable is designed to host almost any web-based app and all the production-ready services your app needs.

- Install Github App, authorize

### [Github Pages](https://pages.github.com/) (frontend)

Github Pages hosts static sites directly from you Github repository with the help of static site generators like Jekyll.

- `rm -rf node_modules/.cache/gh-pages`
- `npm i gh-pages -D`

### [Firebase](https://firebase.google.com/) (frontend)

Google's Firebase is a comprehensive platform that offers hosting, database, authentication, and other backend services. It is ideal for quickly building and scaling web and mobile applications.

- `npm i firebase -D`
- `npm i firebase-tools -g`
- Create project firebase console
- `firebase login`
- `firebase init` or manual file placement
- `firebase deploy`

### [Surge](https://surge.sh/) (frontend)

Surge is a straightforward platform designed for easy and rapid deployment of static websites, providing simple hosting solutions without any complexities.

- `npm install surge --global `
- `surge dist`

### [Fleek](https://fleek.co/) (frontend)

Fleek is a modern platform offering decentralized hosting solutions for static websites and decentralized applications (dApps). It leverages blockchain technology for increased security and reliability.

- Install Github App, authorize

### [Cloudflare Pages](https://pages.cloudflare.com/) (frontend, backend only serverless)

Cloudflare Pages is a powerful platform that offers seamless deployment and hosting for static websites. It also offers automatic scaling, a global edge network, and continuous deployment capabilities.

- Workers & Pages > Create application > Pages > Connect to Git.
- Install Github App, authorize
